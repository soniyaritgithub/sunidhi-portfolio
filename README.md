# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/soniyaritgithub/sunidhi-portfolio.git
```

Go to Project

```bash
cd sunidhi-portfolio
```

Install Packages

```bash
npm install
```

Run

```bash
npm run dev
```

Open

```
http://localhost:3000
```
