# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/soniyaritgithub/sunidhi-portfolio.git
```

Go to project folder

```bash
cd sunidhi-portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Open

```
http://localhost:3000
```
